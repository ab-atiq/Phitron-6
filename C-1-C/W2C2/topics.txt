Statement:
    Given positive integer N and an array "A" of size "N". 
    Also you will be given two values "Index" and "Value". 
    You need to change the value of "Index" to "Value" and then print the array in "reverse way".

Constraints:
1 <= N <= 10^5

Input Format:
    First line will contain N.
    Second line will contain the array A.
    Third line will contain Index and Value.

Output Format:
    Output the final array in reverse order.

input:
5
10 20 30 40 50
4 10

Output:
10 40 30 20 10
